# 🚀 Akshit Sreeperumbuduru — Developer Portfolio

A modern, responsive developer portfolio built using **React, Vite, and Tailwind CSS**, showcasing projects, skills, and experience with a clean UI and smooth animations.

## 🌐 Live Website
https://akshit-sreeperumbuduru-portfolio.vercel.app

---

## 📌 About the Project

This portfolio website is designed to present my work as a **B.Tech Computer Science student** interested in **AI, Machine Learning, and Backend Development**.

It highlights:
- Real-world AI projects (Deepfake Detection, Hospital AI Chatbot)
- Technical skills and tools
- Clean and interactive user experience
- Recruiter-friendly structure

The goal of this project is to create a **professional online presence** that effectively communicates my capabilities and projects.

---

## 🛠️ Tech Stack

- **Frontend:** React + Vite  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **Icons:** React Icons  

---

## ✨ Features

- Responsive design (mobile + desktop)
- Smooth scroll and reveal animations
- Interactive project showcase
- Resume download option
- Clean dark-themed UI with modern effects
